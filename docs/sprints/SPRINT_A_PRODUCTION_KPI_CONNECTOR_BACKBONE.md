# Sprint A - Production KPI Connector Backbone

## Goal

Make Amro's event dashboards connector-first for production reporting.

Manual KPI entry remains available only as an audited fallback/correction path. It is not the production source of truth.

## Delivered

- Added explicit connector sync contract fields to `ConnectorImportJob`:
  - `sync_status`
  - `last_sync_at`
  - `last_sync_rows`
  - `last_sync_error`
  - `last_sync_audit_record_id`
- Added migration `20260704_connector_sync_status`.
- Added connector sync status API:
  - `GET /connector-imports/sync-status`
  - `GET /connector-imports/sync-status?eventId=<eventId>`
- Updated connector import behavior:
  - no fake provider rows are generated when a read-only adapter is missing
  - blocked/failed sync states are persisted with error reason
  - approved imports write exact dry-run rows and persist sync row count/audit id
- Reclassified CSV/customer export imports as `imported`, not `connector`.
- Updated event dashboard API response with:
  - primary source: connector/imported/manual/none
  - manual fallback flag
  - connector job sync status
  - last sync timestamp
  - row count
  - sync errors
- Updated master dashboard API response with portfolio-level data source summary.
- Updated customer-facing UI copy:
  - "Manual KPI Update" is now "Fallback KPI Correction"
  - Event dashboard shows "Production KPI Backbone"
  - Master dashboard shows "Production Data Backbone"

## Not Included

Provider-specific live API clients are intentionally not implemented in this sprint.

Still required in follow-up sprints:

- Meta Marketing API read connector.
- Google Ads / YouTube Ads read connector.
- GoHighLevel lead/opportunity/tag sync connector.
- Formaloo form submission connector.
- WhatsApp/GHL message delivery outcome connector.

## Acceptance Criteria

- Dashboards distinguish connector, approved import, and manual fallback data.
- Manual data is not presented as production source of truth.
- Connector jobs expose auditable sync status and row counts.
- Failed/blocked sync attempts record a reason.
- No fake KPI rows are generated by missing adapters.
